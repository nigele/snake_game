# Classic Snake

Minimal Snake built with React and Vite.

## Run

1. Install dependencies with `npm install`
2. Start the dev server with `npm run dev`
3. Open the local URL shown by Vite, usually `http://localhost:5173/`

## Manual verification

- Arrow keys and `WASD` move the snake
- On-screen controls work on small screens
- The snake cannot reverse directly into itself
- Eating food increases score and snake length
- Wall collisions and self-collisions end the game
- `Pause`, `Resume`, and `Restart` work correctly
